# Hindi Grammar (힌디어 문법)

힌디어 문법의 핵심 규칙들을 정리했습니다.

## Gender Patterns (성별 패턴)

### Basic Endings

| Form | Ending | Korean |
|------|--------|--------|
| 단수형 | 아 (aa) | |
| 복수형 | 에 (e) | |
| 여성형 | 이 (ee) | |

### Conjugation Examples

**남성형**
- karata -> karate (까르떼)

**여성형**
- karata -> karati (까르띠)

## Postpositions (후치사)

힌디어는 전치사 대신 후치사를 사용합니다.

| Hindi | Romanization | Meaning | Example |
|-------|-------------|---------|---------|
| में | men | In | घर में (ghar men) - In the house |
| पर | par | On, ~에 | किताब मेज़ पर है (kitaab mez par hai) - The book is on the table |
| से | se | From | |
| को | ko | To | |

### Comparison

**apapse kam** - less than (당신보다 적게)

## Useful Expressions (유용한 표현)

| Expression | Romanization | Meaning |
|------------|-------------|---------|
| कर दिया गया है | kar diya gaya hai | has been done |
| के लिए | ke liye | for (~위해) |
| करने के लिए | karne ke liye | to do (~하기 위해) |
| करते रहें | karte rahein | keep doing |

### Examples

**2 दिनों के लिए** (2 dino ke liye)
- For 2 days

**सुविधा का दुबारा इस्तेमाल करने के लिए** (suvidha ka dubara istemal karne ke liye)
- To use the service again
