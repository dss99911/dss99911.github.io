## Consonants (자음)

힌디어 자음은 이글자가 저글자인 것 같고, 헷갈리는 부분이 많습니다.

헷갈리는 부분을 헷갈리지 않게, 쉽게 외울 수 있는 방법을 정리했습니다.

### Common Rules (공통사항)

- h 발음의 경우, 공기를 밖으로 내보내기 위에 입을 벌린다. 입을 벌릴때 입 모양이 원이 되므로 글자에도 원 모양이 많이 나온다.

### K Series Consonants (K 계열 자음)

![](/assets/images/posts/language/hindi/-1.png)

**क (k)** - 까 발음
글자에 ㄲ가 있다

![](/assets/images/posts/language/hindi/-2.png)

**ख (kh)**
글자에 ㄲ가 있다
가운데 원모양이 있어 h

![](/assets/images/posts/language/hindi/-3.png)

**ग (g)**
글자에 ㄱ이 있다

![](/assets/images/posts/language/hindi/-4.png)

**घ (gh)**
글자에 ㄱ이 있다
둥근 부분이 있어서 h

### C Series Consonants (C 계열 자음)

![](/assets/images/posts/language/hindi/-5.png)

**च (c)**
글자 안에 c가 있음

![](/assets/images/posts/language/hindi/-6.png)

**छ (ch)**
글자 안에 ch가 있음
원이 있어 h

![](/assets/images/posts/language/hindi/-7.png)

**ज (j)**
글자 안에 J가 있음

![](/assets/images/posts/language/hindi/-8.png)

**झ (jh)**
글자 안에 J가 있고 좀 더 복잡함
इ와 j와의 결합 = jh

---

### T Series Consonants - Heavy (T계열 자음 - 무거운 ㄸ)

혀를 앞니에서 약간 안으로 넣어서 발음

![](/assets/images/posts/language/hindi/-9.png)

**ट (t)**
t랑 비슷하게 아래 굴곡이 있음
따 발음. t 모양과 달리 꼭지가 뒤로 간 이유가 혀의 모양을 형상화한 것 같음

![](/assets/images/posts/language/hindi/-10.png)

**ठ (th)**
ta에서 더 그려서 원을 만듬
h가 있어 원

![](/assets/images/posts/language/hindi/-11.png)

**ड (d)**
글자안에 ㄷ가 있음
혀를 이 에서 약간 뒤에 대고 발음 해서 꼭지가 뒤에 있음

![](/assets/images/posts/language/hindi/-12.png)

**ढ (dh)**
글자안에 ㄷ가 있음
ha발음을 위해 입을 벌려서 원을 만듬. 원이 뒤에 있어 무거운 dha

![](/assets/images/posts/language/hindi/-13.png)

**ण (n)**
ㄴ 발음. 한글의 ㄴ와 달리 혀를 앞니에서 뒤로 넣어서 발음

---

### T Series Consonants - Light (T계열 자음 - 가벼운 ㄸ)

앞니 바로 뒤에 붙여서 발음, 한글의 ㄸ와 동일

![](/assets/images/posts/language/hindi/-14.png)

**त (t)**
t를 눕힌 모양
또는 구부러진 부분이 혀라고 생각하고 ट와는 달리 혀가 혀를 앞으로 보내서 이 바로 뒤에 붙여서 발음한다고 기억

![](/assets/images/posts/language/hindi/-15.png)

**थ (th)**
다른 상단 작대기와 달리 짧게 T를 그리고 있어서, t계열
혀를 앞으로 내미는 모양에서 앞니 바로 뒤에 붙이는 걸 연상. 동그라미 = h

![](/assets/images/posts/language/hindi/-16.png)

**द (d)**
ㄷ이 있음
ड와 구별할 만한 방법이 없음
밑에 대각선이 있으므로, 혀를 대각선으로 세워 이 바로 뒤에 붙인다고 외우기

![](/assets/images/posts/language/hindi/-17.png)

**ध (dh)**
ㄷ 이 있음
h이므로 원이 있음
원이 앞에 있으므로 이 바로 뒤에 붙임

![](/assets/images/posts/language/hindi/-18.png)

**न (n)**
ㄴ 발음

## Pronunciation Rules (발음법)

### Consonant Sounds

| Hindi | Romanization | Korean Equivalent |
|-------|-------------|-------------------|
| स | s | |
| श | sh (strong) | |
| ष | sh | |
| ब | b | |
| ते | | 한글의 ㄸ |
| ट | | 혀를 입천장 앞에 대고 |

### Sound Changes

**d -> r** (중간에서)
- ladaka: 라르까
- padhaana: 빠라h-나

**a -> 으** (중간에 있는 a)
- ladaka: 라르까
- karata: 까르따

**v -> w**
- vah: 워
- danyavaad: 단야왓
