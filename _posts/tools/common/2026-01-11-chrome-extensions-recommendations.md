개발자와 일반 사용자 모두에게 유용한 Chrome 확장 프로그램들을 소개합니다.

## 생산성 도구

### Boomerang for Gmail
Gmail에서 이메일 예약 발송, 리마인더 설정 기능을 제공합니다.

### Save to Pocket
웹 페이지를 저장하고 나중에 읽을 수 있게 해주는 도구입니다.

### WorkFlowy
아웃라인 기반의 노트 앱으로, 생각을 구조화하고 정리하는 데 탁월합니다.

### Evernote Web Clipper
웹 페이지, 기사, PDF 등을 Evernote에 쉽게 저장할 수 있습니다.

## 개발자 도구

### JSON Formatter
JSON 데이터를 보기 좋게 포맷팅하고 하이라이트해줍니다.

### Octotree - GitHub Code Tree
GitHub 저장소를 트리 형태로 탐색할 수 있습니다.
- [Chrome Web Store](https://chrome.google.com/webstore/detail/octotree-github-code-tree/bkhaagjahfmjljalopjnoealnfndnagc)
- 별도 도메인(GitHub Enterprise 등)은 유료

### React Developer Tools
React 애플리케이션 디버깅을 위한 필수 도구입니다.

### JetBrains IDE Support
JetBrains IDE와 Chrome을 연동하여 JavaScript 디버깅을 편리하게 합니다.

### Chrome Extension Source Viewer
다른 확장 프로그램의 소스 코드를 확인할 수 있습니다.

## 자동화 도구

### Tampermonkey
웹 페이지에 커스텀 JavaScript를 주입할 수 있는 강력한 도구입니다.

**유용한 스크립트:**
- [YouTube 광고 숨기기](https://greasyfork.org/scripts/38182-hide-youtube-google-ad/)
- [Local YouTube Downloader](https://greasyfork.org/ko/scripts/369400-local-youtube-downloader)
- [YouTube More Speeds](https://greasyfork.org/ko/scripts/421670-youtube-more-speeds)

### FoxyProxy Standard
프록시 서버를 쉽게 전환하고 관리할 수 있습니다.

## 학습 및 언어 도구

### Google Translate
텍스트 선택 시 바로 번역을 보여줍니다.

### Language Learning with Netflix
Netflix 시청 중 두 개의 자막을 동시에 표시합니다.
- [Language Reactor](https://chrome.google.com/webstore/detail/language-reactor/hoombieeljmmljlkjmnheibnpciblicm)

### Language Learning with YouTube
- [Chrome Web Store](https://chrome.google.com/webstore/detail/language-learning-with-yo/jkhhdcaafjabenpmpcpgdjiffdpmmcjb)

### Grammarly / Ginger
영어 문법 및 맞춤법 검사를 실시간으로 제공합니다.

## 미디어 도구

### Auto Replay for YouTube
YouTube 동영상을 자동으로 반복 재생합니다.

### FireShot
전체 페이지 또는 특정 영역의 스크린샷을 캡처할 수 있습니다.

### Hover Zoom
이미지 위에 마우스를 올리면 확대된 이미지를 보여줍니다.

### Select and Speak
선택한 텍스트를 음성으로 읽어줍니다.

## 보안 및 프라이버시

### HTTPS Everywhere
웹사이트 연결을 자동으로 HTTPS로 전환합니다.

### Web of Trust (WOT)
사이트별 신뢰도 점수를 표시합니다.

### Dark Reader
모든 웹사이트에 다크 모드를 적용합니다.

## 회의 및 협업

### Tactiq for Google Meet
Google Meet, Zoom 등에서 자막을 실시간으로 표시하고 저장합니다.
- [Chrome Web Store](https://chrome.google.com/webstore/detail/tactiq-for-google-meet-an/fggkaccpbmombhnjkjokndojfgagejfb)

## AI 도구

### AIPRM for ChatGPT
ChatGPT에서 사용할 수 있는 프롬프트 템플릿을 제공합니다.

### ChatGPT for Search Engines
검색 엔진 결과 옆에 ChatGPT 응답을 표시합니다.

### YouTube Summary with ChatGPT
YouTube 동영상의 요약을 ChatGPT로 생성합니다.

### WebChatGPT
ChatGPT에 웹 검색 기능을 추가합니다.

### 프롬프트 지니
ChatGPT 대화를 자동으로 번역해주는 한국어 확장 프로그램입니다.

## 기타 유용한 도구

### LINER
웹 페이지와 PDF에 하이라이트를 추가할 수 있습니다.

### Get cookies.txt
웹사이트의 쿠키를 텍스트 파일로 내보낼 수 있습니다.

### Vysor
Chrome에서 Android 기기 화면을 미러링하고 제어할 수 있습니다.

### Draw.io Desktop
브라우저에서 다이어그램을 작성할 수 있는 도구입니다.
