macOS에서 개발과 생산성 향상에 도움이 되는 앱들을 카테고리별로 정리합니다.

## 생산성 도구

### Alfred 4

Mac의 스팟라이트를 대체하는 강력한 런처입니다. 워크플로우를 통해 다양한 자동화가 가능합니다.

### Magnet

윈도우 관리 앱으로, 키보드 단축키로 창을 화면 절반이나 코너에 배치할 수 있습니다.

### Todoist

할 일 관리 앱으로, 크로스 플랫폼을 지원합니다.

### Jumpcut

클립보드 히스토리 관리자입니다. 이전에 복사한 내용을 쉽게 불러올 수 있습니다.

### Bartender 2

메뉴바 아이콘을 정리하여 깔끔한 화면을 유지할 수 있습니다.

## 개발 도구

### JetBrains Toolbox

JetBrains IDE들을 한 곳에서 관리합니다. IntelliJ, WebStorm, PyCharm 등을 쉽게 설치하고 업데이트할 수 있습니다.

### iTerm

Mac 기본 터미널을 대체하는 강력한 터미널 에뮬레이터입니다.

### Sublime Text

빠르고 가벼운 텍스트 에디터입니다.

### Postman

API 테스트 도구로, REST API 개발에 필수적입니다.

### Charles

HTTP/HTTPS 프록시 도구로, 네트워크 디버깅에 유용합니다.

### Wireshark

네트워크 패킷 분석 도구입니다.

## 데이터베이스 도구

### MySQL Workbench

MySQL 데이터베이스 관리 도구입니다.

### Sequel Pro

macOS용 MySQL 클라이언트로, 직관적인 인터페이스를 제공합니다.

### sqlitebrowser

SQLite 데이터베이스를 GUI로 관리할 수 있습니다.

## 디자인 도구

### Sketch

UI/UX 디자인 도구입니다.

### Zeplin

디자인 핸드오프 도구로, 디자이너와 개발자 간 협업에 유용합니다.

## 유틸리티

### iHex

Hex 에디터로, 바이너리 파일을 분석할 때 유용합니다.

### Unarchiver

다양한 압축 형식을 지원하는 압축 해제 도구입니다.

### picatext

스크린샷에서 텍스트를 추출하는 OCR 도구입니다. 정확도는 완벽하지 않지만 가격 대비 실용적입니다.

### Authy

2단계 인증(2FA) 앱으로, 여러 기기 간 동기화를 지원합니다.

## 커뮤니케이션

### Slack

팀 커뮤니케이션 도구입니다.

### KakaoTalk

카카오톡 Mac 버전입니다.

### WhatsApp

WhatsApp Mac 버전입니다.

## 미디어

### Mac Media Player

다양한 비디오 포맷을 지원하는 미디어 플레이어입니다.

### WebTorrent

토렌트 다운로드 및 스트리밍 클라이언트입니다.

## 개발 관련 기타

### Xcode

iOS/macOS 개발을 위한 Apple 공식 IDE입니다.

### Parallels Desktop

Mac에서 Windows를 실행할 수 있는 가상화 도구입니다.

## Android 개발

### Android File Transfer

Mac과 Android 기기 간 파일 전송 도구입니다.

> **팁**: Android Studio에도 동일한 기능이 내장되어 있습니다.

### Vysor

Chrome 앱으로, Android 기기 화면을 Mac에서 미러링할 수 있습니다.

## 클라우드 동기화

### Google Backup and Sync

Google Drive 동기화 클라이언트입니다.

### Evernote

노트 앱으로, 크로스 플랫폼을 지원합니다.

## 쉘/터미널

### Zsh

Bash를 대체하는 강력한 쉘입니다. Oh My Zsh와 함께 사용하면 더욱 편리합니다.

## 키보드 관련

### Karabiner

키보드 매핑을 커스터마이징할 수 있는 도구입니다.

### Haptic Touch Bar

Touch Bar 입력 시 햅틱 피드백을 제공합니다.

- [HapticKey (GitHub)](https://github.com/niw/HapticKey/releases/tag/0.4.6)

## 참고하지 않는 앱들

사용해보았지만 추천하지 않는 앱들:

| 앱 | 이유 |
|----|------|
| Jitouch | 더 이상 업데이트 안 됨 |
| bitbar | 메뉴바 앱, Bartender로 대체 |
| fluid | 웹앱을 데스크톱 앱으로 |
| Paintbrush | 기본 그림판, 기능 부족 |
| ForkLift | 파일 관리자 |
| Translate Tab | 번역 앱 |
| GIMP | 이미지 편집, 사용성 불편 |
