macOS를 더 효율적으로 사용하기 위한 필수 팁과 단축키를 정리합니다.

## 시스템 단축키

### 화면 관련

| 단축키 | 기능 |
|--------|------|
| `Cmd + Ctrl + F` | 풀스크린 모드 전환 |

## Finder 팁

### 숨김 파일 표시/숨기기

```
Cmd + Shift + . (마침표)
```

이 단축키로 숨겨진 파일과 폴더를 토글할 수 있습니다.

### 툴바에 커스텀 버튼 추가

`Cmd` 키를 누른 채로 파일이나 폴더를 툴바로 드래그하면 바로가기 버튼이 추가됩니다.

### Finder에서 자주 사용하는 단축키

| 단축키 | 기능 |
|--------|------|
| `Cmd + Shift + .` | 숨김 파일 토글 |
| `Cmd + Shift + G` | 경로로 이동 |
| `Cmd + T` | 새 탭 열기 |
| `Cmd + N` | 새 창 열기 |
| `Space` | Quick Look |

## 스크린샷

Mac에서 다양한 방식으로 스크린샷을 찍을 수 있습니다.

### 기본 단축키

| 단축키 | 기능 |
|--------|------|
| `Cmd + Shift + 3` | 전체 화면 캡처 |
| `Cmd + Shift + 4` | 영역 선택 캡처 |
| `Cmd + Shift + 4 + Space` | 윈도우 캡처 |
| `Cmd + Shift + 5` | 스크린샷 도구 열기 (Mojave+) |

더 자세한 내용은 [CNET 스크린샷 가이드](https://www.cnet.com/how-to/how-to-take-screenshots-in-macos-mojave/)를 참고하세요.

## 클립보드 이미지 활용

클립보드에 복사된 이미지를 파일로 저장하려면:

1. Preview 앱 실행
2. `File -> New from Clipboard` (또는 `Cmd + N`)
3. 원하는 형식으로 저장

## 화면/데스크톱 관리

### 앱을 특정 데스크톱에 고정

1. Dock에서 앱 아이콘을 우클릭
2. `Options` 선택
3. `This Desktop` 선택

이렇게 하면 해당 앱이 항상 특정 데스크톱에서 열립니다.

## Hyper 터미널

Hyper는 웹 기술로 만들어진 확장 가능한 터미널입니다.

### 추천 플러그인

| 플러그인 | 기능 |
|----------|------|
| `hyper-search` | 터미널 내 검색 |
| `hyper-pane` | 화면 분할 |
| `hyper-material-theme` | Material 디자인 테마 |

설치: [Hyper 플러그인](https://hyper.is/plugins?newest)

## Homebrew

Mac 패키지 관리자인 Homebrew를 사용하면 다양한 도구를 쉽게 설치할 수 있습니다.

설치 및 사용법: [Homebrew 공식 사이트](https://brew.sh/)

## 정리

| 상황 | 해결 방법 |
|------|-----------|
| 숨김 파일 보기 | `Cmd + Shift + .` |
| 클립보드 이미지 저장 | Preview에서 `New from Clipboard` |
| 앱 데스크톱 고정 | Dock에서 우클릭 -> Options -> This Desktop |
| 풀스크린 전환 | `Cmd + Ctrl + F` |

이러한 팁들을 활용하면 Mac 사용이 훨씬 편리해집니다!
