## 비트코인(BTC) 전망

### ★ 이란 공습 + 전쟁 리스크: 최대 하방 촉매 (2/28)

| 항목 | 내용 |
|------|------|
| 이벤트 | **미국-이스라엘 이란 공습 (2/28)** |
| BTC 반응 | **-6% 일간, $63,401까지 하락** |
| 메커니즘 | 전쟁 리스크 → 안전자산 선호 → 달러 강세 → BTC 하방 |
| 유가 영향 | 유가 급등 → 인플레이션 우려 → 금리 인하 지연 → 위험자산 압력 |
| 확대 시나리오 | 이란 보복 → 전면전 → BTC $50K 이하 가능 |
| 해소 시나리오 | 단기 분쟁 종료 → 반등 가능, 그러나 구조적 약세는 유지 |

이란 공습은 BTC에 다층적 하방 압력을 가합니다. (1) 위험자산 회피로 직접 매도 압력, (2) 달러 강세로 간접 하방 압력, (3) 유가 급등 → 인플레이션 재점화 → 금리 인하 지연으로 유동성 환경 악화. **전쟁 리스크가 해소되기 전까지 반등은 제한적**입니다.

### Section 122 15% 관세: 발효 중

| 항목 | 내용 |
|------|------|
| 관세 | **Section 122 15% 관세** |
| 발효일 | **2026년 2월 24일 발효 완료** |
| 영향 | 리스크 자산 전반 하방 압력 (진행 중) |
| BTC 영향 | 달러 강세 → BTC 약세 압력, 글로벌 유동성 위축 |

Section 122 15% 관세에 이란 전쟁 리스크가 겹치면서, 리스크 자산에 대한 하방 압력이 이중으로 작용하고 있습니다.
