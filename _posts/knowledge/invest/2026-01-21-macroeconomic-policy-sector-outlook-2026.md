## ★ Section 122 관세 15% 인상 + USTR 조사 착수 예고

### Section 122 관세 인상 (2/21)

IEEPA 위헌 판결(2/20) 직후 10%로 발동했던 Section 122 보편관세가 **이틀 만에 15%로 인상**되었습니다.

| 항목 | 내용 |
|------|------|
| 인상 일자 | **2월 21일 (금)** |
| 발효 일자 | **2월 24일 (월)** |
| 세율 변경 | **10% → 15%** (법적 상한 도달) |
| 법적 제한 | **최대 150일, 최대 15%** — 법적 상한에 도달, 추가 인상 불가 |
| Section 232 관세 | **반도체·철강 관세 별도 유지** (Section 122와 무관) |

### 환급 소송 이슈

| 항목 | 내용 |
|------|------|
| 환급 대상 금액 | **$1,300억+** (IEEPA 위헌으로 무효화된 관세 징수분) |
| 행정명령 포함 여부 | **미포함** — 행정명령에 환급 관련 조항 없음 |
| 예상 | 대규모 환급 소송 예상 → 재정 부담 |

### USTR 주요 교역국 조사 예고

| 항목 | 내용 |
|------|------|
| 내용 | USTR이 **주요 교역국 대상 무역 관행 조사 착수 예고** |
| 한국 포함 가능성 | **있음** — 주요 교역국에 해당 |
| 의미 | Section 122 만료(150일) 이후를 대비한 **새로운 관세 근거 마련** 시도 |
| 시사점 | 관세 불확실성 장기화 가능 |

### 관세 체제 현황 종합

| 구분 | 근거법 | 세율 | 상태 |
|------|--------|------|------|
| 보편관세 | **Section 122** | **15%** (2/24 발효) | **150일 한시, 상한 도달** |
| 반도체 관세 | **Section 232** | 별도 | **유지** |
| 철강/알루미늄 | **Section 232** | 별도 | **유지** |
| IEEPA 기반 관세 | IEEPA | - | **위헌 무효** |
| 캐나다/멕시코 펜타닐 | IEEPA | - | **별도 소송 진행 중** |
