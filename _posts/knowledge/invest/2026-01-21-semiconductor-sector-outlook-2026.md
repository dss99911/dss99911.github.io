2026년 반도체 섹터가 **기가 사이클(Giga Cycle)**에 본격 진입했습니다. 메모리 시장에서는 **"RAMmageddon"**이라 불리는 역사적 가격 폭등이 진행 중이고, AI 인프라 투자는 **$660-690B**로 전년 대비 2배 규모에 달합니다. **GTC 2026(3/16~19)**을 앞두고 Jensen Huang이 **"세상을 놀라게 할 칩"**을 예고했습니다.

**3월 3~4일 2일간 KOSPI가 -18.4% 폭락(사상 최대, 서킷브레이커 발동)한 뒤, 3월 5일 +9.63%(5,583.9)로 2008년 이후 최대 일일 상승을 기록하며 강력하게 반등했습니다.** 삼성전자 **~195,100원(+11.15%)**, SK하이닉스 **~941,000원(+10.95%)**. 이란 협상 시그널이 반등을 촉발했으며, **폭락이 100% 지정학 리스크였고 펀더멘털은 전혀 훼손되지 않았음이 +11% 반등으로 증명되었습니다.** TrendForce 3/4 보고서에서는 현물가가 계약가를 초과하며 Q2 추가 +20% 상승이 전망되고, 삼성/SK는 구글/MS에 서버 DRAM **60-70% 가격 인상**을 요구하고 있습니다.

## RAMmageddon: 메모리 역사적 폭등 + Q2 추가 +20% + 서버 DRAM 60-70% 인상 요구

### 가격 동향

| 제품 | Q1 변동 | Q2 전망 | 구체 가격 |
|------|---------|---------|----------|
| **범용 DRAM** | **+90-95% QoQ** | **+20% QoQ 컨센서스** | TrendForce 3/4: 현물가 > 계약가 |
| **서버 DRAM (DDR5 블렌디드)** | **+105-110% QoQ** | **60-70% 인상 요구** | 삼성/SK → 구글/MS |
| **64GB RDIMM DDR5** | | | $255(Q3) → $450(Q4) → **$700+(3월)** |
| **DDR5 32GB 키트** | | **$550-600 피크** | $100-200 → $350+ → **Q2 $550-600** |
| **NAND** | **+55-60% QoQ** | | |
| **소비자 SSD 1TB** | | | $45 → **~$90 (2x)** |
| **AI 데이터센터 DRAM** | | | **전체 DRAM 생산의 40% 소비** |

**TrendForce 3/4 보고서**: 현물가(spot)가 계약가(contract)를 초과하는 상황이 발생. Q2 계약 협상이 아직 시작되지 않은 상태에서 **Q2 추가 +20% 상승**이 컨센서스. 삼성/SK하이닉스가 구글·MS에 서버 DRAM **60-70% 가격 인상**을 요구하고 있으며, DDR5 32GB 키트는 Q2에 **$550-600**까지 피크 전망. AI 데이터센터가 전체 DRAM 생산의 **40%**를 소비하면서 RAMmageddon이 **다분기 지속형 슈퍼사이클**임을 확인.

**근본 원인**: HBM 생산이 GB당 **~3배의 웨이퍼 용량**을 소비 → 범용 DRAM 생산 라인 축소(crowding out) → 구조적 공급 부족
**리드타임**: **40주 이상**, 마이크론은 **2028년까지 구조적 부족** 전망

**밸류에이션 관점**: 반등에도 PER ~4.2배 수준 유지(폭락 전 고점 대비 갭 존재). 글로벌 반도체 업종 평균(15-20배) 대비 **73-79% 할인**. 서버 DRAM 60-70% 인상 요구가 현실화되면 실적 추가 개선 → PER 추가 하락 가능.

**3/3 목표가 상향 (폭락 전 기준)**

| 증권사 | 목표가 | OP 전망 |
|--------|--------|---------|
| 키움증권 | **130만원** | OP 170조 |
| 하나증권 | **145만원** | OP 112조 |
| 대신증권 | **145만원** | OP 100.7조 |
| 시티/SK증권 | **140-150만원** | |

**현재가 대비 업사이드**: 939,000원 기준, 목표가 130만원은 **+38.4%**, 145만원은 **+54.4%** 업사이드.

**HBM4 현황**: M15X 팹에서 1b DRAM 기반 HBM4 양산, 초기 월 10K 웨이퍼. CAPEX 중반 30조대, M15X에 20조+ 투자. **NVIDIA 총 HBM4 수요의 약 2/3 공급 확정**으로 HBM4 세대에서도 압도적 지위 유지.

## 투자 전략: 반등이 증명한 테시스, 추가 상승 여력

### 반등 이후 액션 플랜

| 전략 | 내용 |
|------|------|
| **단기 (1-2주)** | GTC 2026(3/16) 카탈리스트 대기. 3/5 반등으로 추세 전환 확인, 추가 상승 기대 |
| **중기 (1-3개월)** | 서버 DRAM 60-70% 인상 현실화 → Q2 실적 서프라이즈 기대. 실적 발표 전 포지션 유지 |
| **장기 (6개월+)** | $975B 기가사이클, AI CAPEX $660-690B 구조적 성장 지속 |

### 투자 근거 강화 요인

1. **폭락=지정학 증명**: 3/5 +11% 반등으로 펀더멘털 무관 확인
2. **PER 여전히 극단적 저평가**: SK하이닉스 PER ~4.2배 (업종 평균 15-20배 대비 73-79% 할인)
3. **서버 DRAM 60-70% 인상 요구**: 삼성/SK → 구글/MS, 실적 추가 개선 동력
4. **현물가 > 계약가**: TrendForce 3/4, Q2 추가 +20% 상승 전망
5. **GTC 2026 (3/16)**: HBM4 쇼케이스 + Isaac GR00T N1 + 3만명 참석
6. **AI CAPEX 불변**: $660-690B, 빅테크 가이던스 변동 없음
7. **AI 데이터센터 DRAM 40%**: 전체 DRAM 생산의 40% 소비 → 구조적 수요

### 매도 트리거 (감시 신호)

1. **DRAM 가격 하락 전환** -- 현재 67-70% 영업마진이 꺾이기 시작할 때
2. **HBM 공급 과잉 신호** -- 3사 동시 증설 가속
3. **빅테크 CAPEX 가이던스 하향** -- AI 투자 모멘텀 둔화
4. **중국 YMTC/CXMT 공급 확장** -- 메모리 센티먼트 악화
5. **Section 122 관세 150일 이후 정책** -- 의회 승인 여부
