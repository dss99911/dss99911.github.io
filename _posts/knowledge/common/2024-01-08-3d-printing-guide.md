3D 프린팅은 디지털 설계를 실제 물체로 만들어내는 혁신적인 기술입니다.

## 3D 프린팅 입문 과정

1. 국내에서 **조립 교육**을 해주는 프린터 구입
2. 3D 프린팅의 **원리 이해**
3. 좀 더 좋은 DIY 3D 프린터를 구입해서 **직접 조립**

## 출력 방식

### FDM(FFF) 방식 - 적층식

가장 보편적이고 저렴한 방식입니다.

#### 장점
- 가격이 저렴
- 재료 다양성
- 유지보수 용이

#### 단점
1. 바닥판이 기울어지면 출력물이 기울어짐
2. 익스트루더에 재료가 제대로 물리지 않으면 두께 불균일
3. 바닥판과 재료의 접착력 부족 시 플로팅 현상
4. 모델에 중공이 있으면 쳐짐 발생 (서포터 필요)
5. 프레임 기울어짐이나 진동 시 문제 발생
6. 적층식 특성상 겉 표면이 매끄럽지 못함
7. 출력 시간이 오래 걸림 (컵 하나 2~3시간)
8. 소음과 냄새 발생

### 기타 방식

- **DLP**: 고해상도, 정밀한 출력
- **SLA**: 레이저 기반, 높은 정밀도
- **SLS**: 분말 소결, 강도 우수

## 익스트루더 종류

### 직결 방식

- 익스트루더와 노즐이 바로 붙어있는 형태
- **장점**: 안정적인 필라멘트 공급
- **단점**: 출력 속도가 상대적으로 느림

### 보우덴 방식

- 익스트루더가 노즐과 떨어져 있음
- **장점**: 직결 대비 **2~3배 빠른 고속 출력**
- **단점**: 초급자에게 어려움

## 필수 용어

| 용어 | 설명 |
|------|------|
| **필라멘트** | 프린터의 잉크와 같은 재료 |
| **익스트루더** | 필라멘트를 밀어내는 장치 |
| **노즐** | 필라멘트가 나오는 부분 |
| **히트베드** | 출력 바닥판 (온도 조절) |
| **서포터** | 중공 부분을 지탱하는 구조물 |

## 추천 프린터 선택 기준

```
출력 방식: 직결 (간단한 방식, 입문자에게 적합)
출력 사이즈: 200 × 200 × 120mm (큰 것이 유리)
히트베드: 필수 (선택사항으로 되어 있어도 필수)
교육: 본사 교육 추천
```

### 추천 브랜드

- **크리메이커**: 가성비 좋음, 성능 우수
