# KT 소액결제 차단/한도설정 완벽 가이드

---

## 소액결제 차단이 필요한 이유

- 스미싱 피해 예방
- 자녀 무단 결제 방지
- 해킹 피해 최소화
- 불필요한 과금 방지

---

## 차단/한도설정 방법

### 방법 1: 마이케이티(My KT) 앱

```
My KT 앱 로그인
    ↓
마이 → 요금/서비스
    ↓
요금조회 → 휴대폰결제 내역
    ↓
휴대폰결제 → 한도변경
    ↓
원하는 한도 선택 (0원 = 차단)
    ↓
즉시 반영
```

### 방법 2: KT 홈페이지

1. [KT 홈페이지](https://www.kt.com) 접속
2. 로그인 → 마이페이지
3. 요금/서비스 → 휴대폰결제 → 한도변경
4. 한도 설정 (0원~100만원)

### 방법 3: 고객센터 전화 (114)

1. 휴대폰에서 **114** 전화
2. 상담원 연결
3. 본인 인증
4. 소액결제 한도 변경 요청

## 차단 서비스 종류

### 1. 일반차단 서비스

| 특징 | 내용 |
|------|------|
| **해제 가능** | 본인이 직접 한도 변경 가능 |
| **설정 방법** | 앱/웹/전화 모두 가능 |
| **용도** | 임시 차단, 한도 조절 시 |

### 2. 원천차단 서비스

| 특징 | 내용 |
|------|------|
| **해제 불가** | 비대면 해제 불가능 |
| **설정 방법** | 고객센터/앱/매장 |
| **해제 방법** | KT플라자 방문만 가능 |
| **용도** | 완전한 피해 예방 |

## 원천차단 신청 방법

### 전화 신청

| 대상 | 연락처 |
|------|--------|
| **KT 이용자** | 080-001-0126 |
| **KT망 알뜰폰** | 해당 알뜰폰사 고객센터 |
| **전담 고객센터** | 080-722-0100 |

### 앱/매장 신청
- My KT 앱에서 원천차단 신청
- KT플라자 방문 신청

**주의**: 원천차단 해제는 **KT플라자 방문**만 가능!

## 해제 방법

### 일반차단 해제

```
My KT 앱/웹 로그인
    ↓
한도변경 메뉴
    ↓
원하는 한도로 상향 조정
    ↓
즉시 적용
```

### 원천차단 해제

```

    ↓
신분증 지참
    ↓
본인 확인
    ↓
해제 신청
```

## 피해 예방 루틴

### 일일 체크

- [ ] 출처 불명 문자/링크 클릭 금지
- [ ] 앱 설치 시 권한 확인
- [ ] 모르는 번호 콜백 주의

### 주간 체크

- [ ] My KT 앱에서 결제 내역 확인
- [ ] 이상 결제 발견 시 즉시 신고

### 설정 체크

- [ ] 소액결제 한도 적정 수준으로 설정
- [ ] 미사용 시 0원 차단 권장
- [ ] 원천차단 고려 (해제 필요 시 매장 방문)

---

## 피해 발생 시 대응

### 즉시 조치

1. **결제 차단**: My KT 앱에서 한도 0원으로 변경
2. **통신사 신고**: 114 또는 고객센터
3. **경찰 신고**: 사이버수사대 (182)
4. **금융감독원**: 1332

### 환불 요청

1. 결제 내역 확인 (My KT 앱)
2. 해당 콘텐츠 제공업체에 환불 요청
3. 통신사 분쟁 조정 신청

## 자주 묻는 질문

### Q1. 차단하면 모든 휴대폰 결제가 막히나요?
**A**: 네. 소액결제, 콘텐츠 이용료 등 모든 휴대폰 결제가 차단됩니다.

### Q2. 차단/해제는 몇 번까지 가능한가요?
**A**: 횟수 제한 없이 필요할 때마다 설정/해제 가능합니다. (일반차단 기준)

### Q3. 차단해도 요금제는 유지되나요?
**A**: 네. 기본 요금제와 무관하게 소액결제만 차단됩니다.

### Q4. 가족 회선도 함께 차단할 수 있나요?
**A**: 각 회선별로 개별 설정해야 합니다.

### Q5. 해외에서도 차단/해제 가능한가요?
**A**: 앱/웹을 통해 가능합니다. 단, 원천차단 해제는 국내 매장 방문 필요.

## 스미싱 피해 실제 사례

소액결제를 통한 피해는 주로 스미싱(SMS + 피싱)을 통해 발생합니다. 실제 자주 발생하는 사례를 알아두면 예방에 도움이 됩니다.

### 사례 1: 택배 사칭 문자

```
[CJ대한통운] 배송 불가 지역입니다. 주소 확인 바랍니다.
http://악성링크.com
```

택배 회사를 사칭하여 악성 앱 설치를 유도합니다. 앱이 설치되면 인증번호 문자를 가로채 소액결제를 진행합니다.

### 사례 2: 정부 기관 사칭

```
[국민건강보험] 건강검진 결과 확인하기
http://악성링크.com
```

건강보험공단, 국세청, 법원 등을 사칭합니다. 공공기관은 문자로 앱 설치를 요구하지 않습니다.

### 사례 3: 지인 사칭

```
엄마 나 폰이 고장나서 수리 맡겼어
이 번호로 문자해줘. 급한 게 있어
```

가족이나 지인을 사칭하여 개인정보를 요구하거나 송금을 유도합니다.

### 스미싱 문자 구별법

| 특징 | 정상 문자 | 스미싱 문자 |
|------|----------|-----------|
| **링크** | 공식 도메인 (kt.com 등) | 축약URL, 이상한 도메인 |
| **내용** | 구체적인 정보 포함 | 급박함을 강조 |
| **앱 설치** | 공식 스토어로 안내 | 직접 APK 설치 유도 |
| **개인정보** | 요구하지 않음 | 주민번호, 카드번호 등 요구 |

## 통신사별 소액결제 차단 비교

KT뿐만 아니라 다른 통신사에서도 소액결제 차단이 가능합니다.

| 기능 | KT | SKT | LG U+ |
|------|-----|-----|-------|
| **차단 앱** | My KT | T world | 당겨요 |
| **고객센터** | 114 | 114 | 114 |
| **원천차단** | KT플라자 방문 해제 | SKT 매장 방문 해제 | LG U+ 매장 방문 해제 |
| **한도 설정** | 0~100만원 | 0~100만원 | 0~100만원 |
| **즉시 반영** | O | O | O |

### 알뜰폰(MVNO) 사용자

알뜰폰 사용자는 해당 알뜰폰 사업자의 고객센터를 통해 소액결제를 관리해야 합니다. 다만 원천차단은 기반 통신사(KT/SKT/LG U+)의 매장 방문이 필요할 수 있습니다.

## 자녀 보호를 위한 추가 설정

어린이나 청소년의 스마트폰에서는 소액결제 차단 외에도 추가적인 보호 설정을 권장합니다.

### 앱 설치 제한

- **앱 스토어 인증**: Google Play 또는 App Store에서 결제 시 비밀번호/생체인증 필수 설정
- **출처 알 수 없는 앱**: "알 수 없는 출처에서 앱 설치" 기능 비활성화
- **자녀 보호 모드**: Google Family Link 또는 Apple 스크린 타임 활용

### 결제 알림 설정

소액결제가 발생하면 즉시 알림을 받을 수 있도록 설정합니다:

1. My KT 앱 → 알림 설정 → 결제 알림 ON
2. 문자 수신 → 소액결제 인증번호 문자 확인
3. 통장 자동 알림 → 출금 시 즉시 알림

## 소액결제 관련 법적 권리

### 미성년자 결제 취소

미성년자가 법정대리인의 동의 없이 소액결제를 한 경우, 민법 제5조에 따라 취소할 수 있습니다. 다만, 미성년자가 성인인 것처럼 속이거나 법정대리인의 동의가 있는 것처럼 한 경우에는 취소가 어려울 수 있습니다.

### 사기 피해 시 환불 절차

1. **결제 내역 확보**: My KT 앱에서 상세 결제 내역 캡처
2. **경찰 신고**: 사이버수사대(182)에 피해 신고
3. **통신사 이의 제기**: 피해 사실을 알린 후 이의 신청
4. **콘텐츠 제공업체 환불 요청**: 결제된 콘텐츠 업체에 직접 환불 요청
5. **금감원 분쟁 조정**: 해결되지 않으면 금융감독원(1332)에 분쟁 조정 신청
