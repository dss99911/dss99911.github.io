# 유심(USIM) 교체 완벽 가이드

---

## 유심 교체 전 백업 체크리스트

### 필수 백업 항목

| 항목 | 백업 방법 |
|------|----------|
| **유심 저장 연락처** | 휴대폰 설정 → 연락처 → 연락처 가져오기/내보내기 → 단말기로 복사 |
| **교통카드 잔액** | 티머니 앱 → 잔액 환불 신청 |
| **후불 교통카드** | 앱에서 삭제 처리 후 유심 교체 |

### 백업 불필요 (휴대폰에 저장됨)

- 사진, 동영상
- 앱 및 앱 데이터
- 문자 메시지
- 휴대폰 저장 연락처

**참고**: 유심을 바꿔도 전화번호는 그대로 유지됩니다!

---

## 유심 교체 기본 절차

```
① 휴대폰 전원 끄기 (권장)
    ↓
② 유심 트레이 열기 (측면 구멍에 유심 핀 삽입)
    ↓
③ 기존 유심 제거
    ↓
④ 새 유심 장착 (방향 맞춰서 홈에 끼우기)
    ↓
⑤ 트레이 다시 삽입
    ↓
⑥ 전원 켜기
    ↓
⑦ 네트워크 연결 확인
```

**주의**: 유심 방향이 틀리면 인식이 안 됩니다!

## 통신사별 무료 교체 안내 (2025년)

### SK텔레콤 (SKT)

| 항목 | 내용 |
|------|------|
| **무료 교체 대상** | 2025.04.18 24시 기준 SKT 이용 고객 |
| **교체 장소** | 전국 T월드 매장 |
| **횟수** | 1회 한정 |
| **eSIM 변경** | 가능 (최신 기종 권장) |

**팁**: 유심 재고 부족 시 eSIM으로 변경하는 것이 빠릅니다.

### KT

| 항목 | 내용 |
|------|------|

| **택배 셀프교체** | 2025.11.11부터 병행 |
| **eSIM 변경** | 방문 예약 후 가능 |

| 항목 | 내용 |
|------|------|

## eSIM 셀프 교체 방법

### 준비물
- Wi-Fi 연결 필수
- eSIM 지원 단말기

### 삼성 스마트폰

```
설정 → 연결 → SIM 관리자 → SIM 선택 → eSIM으로 전환
```

### 아이폰

```
설정 → 셀룰러 → eSIM으로 전환
```

## eSIM vs USIM 비교

| 구분 | USIM | eSIM |
|------|------|------|
| **형태** | 물리적 칩 | 내장형 (칩 없음) |
| **교체 방법** | 매장 방문 또는 배송 | 온라인 즉시 설치 |
| **기기 변경** | 유심 옮기면 됨 | 새 기기에서 재다운로드 필요 |
| **듀얼심** | 슬롯 필요 | 물리 유심과 함께 사용 가능 |
| **분실 위험** | 있음 | 없음 |

### eSIM 장점
- 매장 방문/배송 대기 불필요
- 여러 회선을 저장해두고 전환 가능
- 물리적 분실 위험 없음

### eSIM 단점
- 기기 변경 시 재다운로드 필요 (비용 발생 가능)
- 일부 구형 기기 미지원
- 해외 로밍 시 호환성 확인 필요

## 주의사항 체크리스트

### 교체 전

- [ ] 유심에 저장된 연락처 → 휴대폰으로 복사
- [ ] 티머니 선불 잔액 → 앱에서 환불 신청
- [ ] 후불 교통카드 → 앱에서 삭제 처리
- [ ] 유심 핀 준비 (또는 클립)

### 교체 시

- [ ] 전원 끄고 작업 (데이터 손상 방지)
- [ ] 유심 방향 확인 (노치 위치)
- [ ] 트레이 완전히 삽입

### 교체 후

- [ ] 네트워크 연결 확인
- [ ] 전화/문자 테스트
- [ ] 모바일 데이터 확인
- [ ] 교통카드 재등록 (필요시)

## 자주 묻는 질문

### Q1. 유심 교체하면 데이터 날아가나요?
**A**: 사진, 앱, 문자 등은 휴대폰에 저장되어 있어서 유심 교체와 무관합니다. 단, 유심에 저장된 연락처는 미리 백업 필요.

### Q2. 전화번호 바뀌나요?
**A**: 아니요. 유심만 바꾸면 기존 전화번호 그대로 사용합니다.

### Q3. 직접 교체해도 되나요?
**A**: 네. 유심 트레이만 열 수 있으면 셀프 교체 가능합니다.

### Q4. eSIM으로 바꾸면 유심 슬롯은 어떻게 되나요?
**A**: 비어 있게 됩니다. 나중에 듀얼심으로 활용하거나, 해외 로밍 유심 장착 시 사용 가능.
