# 2025 상생페이백 완벽 가이드

상생페이백은 2025년 9월~11월 카드 사용액이 2024년 월평균보다 늘어나면, 증가분의 20%를 디지털 온누리상품권으로 돌려받는 정부 환급 제도입니다.

---

## 환급 구조

### 기본 계산법

```
환급액 = (2025년 월 사용액 - 2024년 월평균 사용액) × 20%
```

**예시:**
- 2024년 월평균 카드 사용액: 100만원
- 2025년 9월 사용액: 130만원
- 증가분: 30만원
- **환급액: 6만원** (30만원 × 20%)

### 환급 한도

| 구분 | 한도 |
|------|------|
| **월 최대** | 10만원 |
| **총 최대 (3개월)** | 30만원 |

---

## 신청 대상 및 기간

| 항목 | 내용 |
|------|------|
| **대상** | 2024년 신용/체크카드 사용 실적이 있는 만 19세 이상 국민 및 외국인 |
| **신청 기간** | 2025.09.15(월) 09:00 ~ 2025.11.30(일) 24:00 |
| **신청 방법** | 상생페이백 전용 누리집 (휴대폰 본인 인증) |

---

## 지급일

| 사용월 | 지급일 |
|--------|--------|
| 2025년 9월 사용분 | 10월 15일 |
| 2025년 10월 사용분 | 11월 15일 |
| 2025년 11월 사용분 | 12월 15일 |

**유효기간**: 지급일로부터 5년간 사용 가능

## 소비액 인정 사용처

### 인정되는 곳

| 업종 | 예시 |
|------|------|
| **전통시장** | 재래시장, 상설시장 |
| **음식점** | 동네 식당, 카페 |
| **미용실** | 헤어샵, 네일샵 |
| **프랜차이즈** | 중소형 프랜차이즈 가맹점 |
| **슈퍼마켓** | 연 매출 30억 원 이하 중형 슈퍼 |
| **제과점** | 동네 빵집, 중소형 체인 |

### 결제 수단

| 인정 | 불인정 |
|------|--------|

| 삼성페이 (오프라인) | 간편결제 (카카오페이, 네이버페이 등) |
| 애플페이 (오프라인) | QR결제 |
| | 상품권 결제 |

## 환급금 사용처 (디지털 온누리상품권)

환급받은 상생페이백은 **디지털 온누리상품권** 형태로 지급됩니다.

- 전통시장
- 소상공인 상점
- 골목형 상점가
- **전국 약 13만 개 가맹점**에서 사용 가능

---

## 자주 막히는 오류 및 해결법

### 1. 신청이 안 되는 경우

| 원인 | 해결 |
|------|------|
| 2024년 카드 실적 없음 | 2024년 신용/체크카드 사용 이력 필요 |
| 만 19세 미만 | 성인만 신청 가능 |
| 본인 인증 실패 | 휴대폰 명의 확인, 통신사 인증 재시도 |

### 2. 환급이 적게 들어온 경우

| 원인 | 해결 |
|------|------|
| 불인정 업종에서 결제 | 소상공인 매장 위주로 결제 |
| 간편결제 사용 | 신용/체크카드 직접 결제로 변경 |
| 온라인 결제 | 오프라인 매장에서 결제 |

### 3. 환급금 사용이 안 되는 경우

| 원인 | 해결 |
|------|------|
| 가맹점이 아닌 곳 | 온누리상품권 가맹점 확인 |
| 앱 오류 | 앱 업데이트 또는 재설치 |
| 잔액 부족 | 잔액 확인 후 금액 내 결제 |

## 상생소비복권 추가 혜택

상생페이백 신청 시 **자동 응모**되는 복권 이벤트입니다.

### 응모 조건
- 상생페이백 신청 완료
- 5만 원 소비당 복권 1장 (월 최대 10장)

### 당첨금

| 등수 | 상금 |
|------|------|
| 1등 | 2,000만원 |
| 2등~5등 | 다양한 경품 및 상금 |
