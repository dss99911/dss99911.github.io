# KT 소액결제 차단/한도설정 완벽 가이드

---

## 소액결제 차단이 필요한 이유

- 스미싱 피해 예방
- 자녀 무단 결제 방지
- 해킹 피해 최소화
- 불필요한 과금 방지

---

## 차단/한도설정 방법

### 방법 1: 마이케이티(My KT) 앱

```
My KT 앱 로그인
    ↓
마이 → 요금/서비스
    ↓
요금조회 → 휴대폰결제 내역
    ↓
휴대폰결제 → 한도변경
    ↓
원하는 한도 선택 (0원 = 차단)
    ↓
즉시 반영
```

### 방법 2: KT 홈페이지

1. [KT 홈페이지](https://www.kt.com) 접속
2. 로그인 → 마이페이지
3. 요금/서비스 → 휴대폰결제 → 한도변경
4. 한도 설정 (0원~100만원)

### 방법 3: 고객센터 전화 (114)

1. 휴대폰에서 **114** 전화
2. 상담원 연결
3. 본인 인증
4. 소액결제 한도 변경 요청

|------|------|

| **50만원** | 중간 한도 |

## 차단 서비스 종류

### 1. 일반차단 서비스

| 특징 | 내용 |
|------|------|
| **해제 가능** | 본인이 직접 한도 변경 가능 |
| **설정 방법** | 앱/웹/전화 모두 가능 |
| **용도** | 임시 차단, 한도 조절 시 |

### 2. 원천차단 서비스

| 특징 | 내용 |
|------|------|
| **해제 불가** | 비대면 해제 불가능 |
| **설정 방법** | 고객센터/앱/매장 |
| **해제 방법** | KT플라자 방문만 가능 |
| **용도** | 완전한 피해 예방 |

---

## 원천차단 신청 방법

### 전화 신청

| 대상 | 연락처 |
|------|--------|
| **KT 이용자** | 080-001-0126 |
| **KT망 알뜰폰** | 해당 알뜰폰사 고객센터 |
| **전담 고객센터** | 080-722-0100 |

### 앱/매장 신청
- My KT 앱에서 원천차단 신청
- KT플라자 방문 신청

**주의**: 원천차단 해제는 **KT플라자 방문**만 가능!

---

## 해제 방법

### 일반차단 해제

```
My KT 앱/웹 로그인
    ↓
한도변경 메뉴
    ↓
원하는 한도로 상향 조정
    ↓
즉시 적용
```

### 원천차단 해제

```
KT플라자 방문 (필수)
    ↓
신분증 지참
    ↓
본인 확인
    ↓
해제 신청
```

---

## 피해 예방 루틴

### 일일 체크

- [ ] 출처 불명 문자/링크 클릭 금지
- [ ] 앱 설치 시 권한 확인
- [ ] 모르는 번호 콜백 주의

### 주간 체크

- [ ] My KT 앱에서 결제 내역 확인
- [ ] 이상 결제 발견 시 즉시 신고

### 설정 체크

- [ ] 소액결제 한도 적정 수준으로 설정
- [ ] 미사용 시 0원 차단 권장
- [ ] 원천차단 고려 (해제 필요 시 매장 방문)

---

## 피해 발생 시 대응

### 즉시 조치

1. **결제 차단**: My KT 앱에서 한도 0원으로 변경
2. **통신사 신고**: 114 또는 고객센터
3. **경찰 신고**: 사이버수사대 (182)
4. **금융감독원**: 1332

### 환불 요청

1. 결제 내역 확인 (My KT 앱)
2. 해당 콘텐츠 제공업체에 환불 요청
3. 통신사 분쟁 조정 신청

## 자주 묻는 질문

### Q1. 차단하면 모든 휴대폰 결제가 막히나요?
**A**: 네. 소액결제, 콘텐츠 이용료 등 모든 휴대폰 결제가 차단됩니다.

### Q2. 차단/해제는 몇 번까지 가능한가요?
**A**: 횟수 제한 없이 필요할 때마다 설정/해제 가능합니다. (일반차단 기준)

### Q3. 차단해도 요금제는 유지되나요?
**A**: 네. 기본 요금제와 무관하게 소액결제만 차단됩니다.

### Q4. 가족 회선도 함께 차단할 수 있나요?
**A**: 각 회선별로 개별 설정해야 합니다.

### Q5. 해외에서도 차단/해제 가능한가요?
**A**: 앱/웹을 통해 가능합니다. 단, 원천차단 해제는 국내 매장 방문 필요.
